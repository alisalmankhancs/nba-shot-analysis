# NBA Shot Efficiency Analysis

## Overview
This project analyzes NBA shot data to explore how shot location, distance, and shot type impact scoring efficiency. The goal is to use real game data to answer practical basketball questions through data analysis and visualization.

## Questions Explored
- How does field goal percentage vary by shot distance?
- Which areas of the court are most efficient?
- How do different shot types compare in efficiency?

## Data
The dataset used contains detailed NBA shot attempt information, including shot location, distance, result, and shot type.

## Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## Status
This project is currently in progress.
